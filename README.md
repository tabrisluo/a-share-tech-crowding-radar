# 📡 A股科技板块"缩圈雷达" V4.5

> **A-share Technology Sector Crowding & Crash Risk Monitoring System**

## 一句话介绍

一个基于**五维加权评分模型**的A股科技板块（TMT/AI）拥挤度风险监控系统，通过量化资金集中度、筹码结构、杠杆水平和全球联动性，输出四级风险信号（绿/黄/橙/红），帮助投资者识别科技板块泡沫破裂前的预警信号。

---

## 🎯 产品定位

| 维度 | 说明 |
|------|------|
| **目标用户** | A股科技板块投资者（关注TMT/AI/光模块/半导体方向的散户及量化交易者） |
| **核心价值** | 将"主观经验判断"（如"科技股太热了"）转化为**可量化、可回测、可执行**的风险评分 |
| **使用场景** | 每日收盘后评估拥挤度、重大事件前判断风险、科技仓位管理的决策辅助 |
| **差异化** | 不同于传统的PE/PB估值模型，专注于**微观交易数据**（成交额、换手率、融资盘）和**全球映射**（美股SOX/NVDA联动） |

---

## 🏗️ 架构设计

### 五维评分模型（V4.5）

```
Score = S_vol × 0.35 + S_turn × 0.25 + S_margin × 0.20 + S_us × 0.10 + S_div × 0.10
```

| 维度 | 权重 | 指标 | 含义 |
|------|------|------|------|
| **资金虹吸** | 35% | TMT四大ETF成交额历史分位数 | 资金对科技板块的"抽血"程度 |
| **筹码松动** | 25% | AI龙头平均换手率历史分位数 | 高位放量=早期获利盘撤离 |
| **杠杆脆弱** | 20% | 融资买入额占比历史分位数 | 杠杆资金是踩踏风险的"易燃物" |
| **全球映射** | 10% | SOX/NVDA是否跌破50日均线 | A股AI估值锚定在美股景气预期 |
| **量价背离** | 10% | 放量滞涨检测 | 经典技术见顶信号 |

### 四级信号体系

| 评分 | 信号 | 操作铁律 | 出现频率 |
|------|------|---------|---------|
| < 45 | 🟢 绿灯 | 可建仓，单股≤20% | ~58% |
| 45-65 | 🟡 黄灯 | 停止加杠杆，停止追高 | ~27% |
| 65-80 | 🟠 橙灯 | 考虑减仓至40-60% | ~13% |
| ≥ 80 | 🔴 红灯 | 仓位降至20%以下 | ~1-2% |

## 🔧 使用方法

### Web应用（推荐日常使用）

1. 打开 https://i35n65fscun26.ok.kimi.link
2. 调节三个滑块（TMT成交占比/龙头换手率/融资占比）
3. 根据需要打开两个开关（美股破位/量价背离）
4. 实时查看评分仪表盘、雷达图和信号判断

### Python脚本（批量计算/自动化）

```bash
# 安装依赖
pip install numpy pandas

# 命令行计算评分
python scripts/calculate_crowding_score.py \
  --tmt-ratio 38.5 \
  --turnover 2.75 \
  --margin 10.5 \
  --us-break true \
  --divergence false
```

### 作为Python模块导入

```python
from scripts.calculate_crowding_score import calculate_v45_score

result = calculate_v45_score(
    tmt_ratio=38.5,
    dragon_turnover=2.75,
    margin_ratio=10.5,
    us_break_50ma=True,
    price_volume_divergence=False,
)
print(f"Score: {result['score']}, Signal: {result['signal']}")
```

## ⚠️ 重要声明

1. **本工具仅供研究参考，不构成投资建议**
2. **红灯样本量不足**：当前可用数据区间仅约481天，红灯样本仅7-18天，统计显著性有限
3. **ETF成交额是代理变量**：四大TMT ETF的成交额合计不等同于申万TMT行业全部个股的真实总成交额
4. **参数未经完整周期验证**：权重分配和阈值分界基于理论推导和有限数据优化，未经完整牛熊周期检验
5. **橙灯"正收益陷阱"**：历史数据显示橙灯后市场仍可能上涨2-6%，减仓的核心目的是风险管理，不是方向预测
